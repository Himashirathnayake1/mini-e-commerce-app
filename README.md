# 🛍️ Mini E-Commerce Flutter App

A modern Flutter e-commerce UI app built with Provider state management and local/mock data, developed as part of a Flutter internship assessment task.

The app includes product listing, product details, a fully functional cart system, and local favorites storage using SharedPreferences.

---

## ✨ Features

-  Splash Screen
-  Product Home Screen (Grid UI)
-  Search by Product Name
-  Product Details Screen 
-  Cart System (Provider state management)
-  Quantity increase / decrease
-  Remove from cart
-  Total price calculation
-  Favorites system (local storage using SharedPreferences)
-  Profile Screen (static UI)
-  Modern UI with shadows, gradients & responsive design
-  Fully responsive layout using `flutter_screenutil`

---



## 📁 Project Structure

```text
lib/
│
├── data/              # Mock product data
├── models/            # Product & Cart models
├── providers/         # Cart & Favorites providers
├── screens/           # All UI screens
├── widgets/           # Reusable widgets 
└── main.dart          # App entry point
```

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Himashirathnayake1/mini-e-commerce-app.git
```

### 2. Move into the project folder
```bash
cd mini-e-commerce-app
```

### 3. Install dependencies
```bash
flutter pub get
```

### 4. Run the app
```bash
flutter run
```
